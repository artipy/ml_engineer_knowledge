# 1.8 Linear Algebra Refresher

> [!summary]
> Ключевые операции линейной алгебры: умножение векторов, матриц и специальные матрицы. В NumPy всё это делается через метод `.dot()` и `np.linalg`.

## Векторы

В линейной алгебре вектор — это **column vector** (столбец). В NumPy вектор по умолчанию одномерный (`shape: (n,)`).

```python
u = np.array([2, 4, 5, 6])
v = np.array([1, 0, 0, 2])

u + v      # Vector addition — поэлементно: [3, 4, 5, 8]
2 * u      # Scalar multiplication: [4, 8, 10, 12]
u * v      # Поэлементное (НЕ dot product): [2, 0, 0, 12]
```

> [!warning]
> `u * v` в NumPy — это **поэлементное** умножение, а не dot product. Результаты выглядят похоже, но это разные операции.

## Vector-vector multiplication (Dot product)

**Dot product** (скалярное произведение, inner product) — результат одно число:

```
dot(u, v) = u[0]*v[0] + u[1]*v[1] + ... + u[n]*v[n]
```

```python
def vector_vector_multiplication(u, v):
    assert u.shape[0] == v.shape[0]
    result = 0
    for i in range(u.shape[0]):
        result += u[i] * v[i]
    return result

vector_vector_multiplication(u, v)  # 14
u.dot(v)                            # 14 — то же самое
```

> [!tip]
> Для записи dot product первый вектор должен быть **row vector**, второй — **column vector**. Преобразование: операция **transpose**.

## Matrix-vector multiplication

Умножение матрицы `U` на вектор `v` — dot product каждой строки `U` с вектором `v`.

> [!important]
> Условие: `U.shape[1] == v.shape[0]` (кол-во столбцов матрицы = кол-ву элементов вектора).

```python
U = np.array([[2,4,5,6], [1,2,1,2], [3,1,2,1]])  # shape (3, 4)
v = np.array([1,0,0,2])                            # shape (4,)

U.dot(v)  # [14, 5, 5] — вектор из 3 элементов
```

## Matrix-matrix multiplication

Матрица `U` умножается на каждый столбец матрицы `V` по очереди (серия matrix-vector умножений).

> [!important]
> Условие: `U.shape[1] == V.shape[0]`.

```python
V = np.array([[1,1,2],[0,0.5,1],[0,2,1],[2,1,0]])  # shape (4, 3)

U.dot(V)
# [[14. , 20. , 13. ],
#  [ 5. ,  6. ,  5. ],
#  [ 5. ,  8.5,  9. ]]
```

## Identity matrix (Единичная матрица)

**Квадратная матрица** с единицами на диагонали и нулями везде остальном. Роль числа "1" в линейной алгебре: `A.dot(I) == A`.

```python
I = np.eye(3)
V.dot(I)  # → V без изменений
```

## Matrix inverse (Обратная матрица)

`A^-1` — матрица, при умножении на которую исходная матрица даёт `I`.

> [!important]
> Обратную матрицу можно вычислить только для **квадратных матриц** (rows == cols).

```python
Vs = V[[0, 1, 2]]         # берём квадратную часть (3×3)
Vs_inv = np.linalg.inv(Vs)
Vs_inv.dot(Vs)             # → Identity matrix
```

> [!tip]
> `np.linalg.inv()` будет ключевым инструментом при реализации **Linear Regression**.

## Шпаргалка: NumPy-функции

| Операция | NumPy |
|---|---|
| Dot product (векторы) | `u.dot(v)` |
| Matrix-vector | `U.dot(v)` |
| Matrix-matrix | `U.dot(V)` |
| Identity matrix | `np.eye(n)` |
| Inverse matrix | `np.linalg.inv(A)` |
