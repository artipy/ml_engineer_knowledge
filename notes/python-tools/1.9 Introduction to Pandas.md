# 1.9 Introduction to Pandas

> [!summary]
> **Pandas** — главная библиотека для работы с табличными данными (**tabular data**). Стандартный импорт: `import pandas as pd`. Под капотом использует NumPy; получить массив можно через `.values`.

## Основные структуры данных

| Структура | Описание |
|---|---|
| **DataFrame** | Таблица с данными — главная абстракция библиотеки |
| **Series** | Один столбец датафрейма |

## Создание и первичный просмотр

```python
# Из списка списков
df = pd.DataFrame(data=data, columns=columns)

# Из list of dictionaries (ключи → названия колонок)
df = pd.DataFrame(data)

df.head()    # первые 5 строк
df.head(2)   # первые N строк

df.to_dict(orient='records')  # DataFrame → list of dicts
```

> [!tip]
> При создании из `list of dictionaries` ключи словарей автоматически становятся названиями колонок.

## Работа со столбцами

```python
df.Make              # доступ через точку
df['Make']           # доступ через скобки (обязателен при пробелах в названии)
df[['Make', 'Model', 'MSRP']]  # несколько колонок → двойные скобки

df['id'] = [1, 2, 3, 4, 5]   # создать/заменить колонку
del df['id']                   # удалить колонку
```

> [!warning]
> Для доступа к колонкам с пробелами или спецсимволами в названии — только квадратные скобки: `df['Engine HP']`.

## Индексы и доступ к строкам

> [!important]
> По умолчанию индекс — `RangeIndex` (0, 1, 2, ...). Можно заменить на произвольные значения.

| Метод | Описание |
|---|---|
| `df.loc[1]` | Доступ по **явному индексу** (числа или строки) |
| `df.iloc[1]` | Доступ по **порядковому номеру** (всегда от 0) |
| `df.reset_index(drop=True)` | Сбросить индекс к sequential (0, 1, 2, ...) |

```python
df.index = ['a', 'b', 'c', 'd', 'e']  # установить строковый индекс
df.loc[['b', 'c']]    # по явным индексам 'b' и 'c'
df.iloc[[1, 2]]       # по позициям 1 и 2 (те же строки)
df.reset_index(drop=True, inplace=True)  # вернуть к числовому
```

## Операции с данными и фильтрация

```python
df['Engine HP'] * 2       # поэлементная операция (NaN игнорируется)

df[df['Year'] >= 2015]    # boolean filtering

# Объединение условий через &
df[(df['Make'] == 'Nissan') & (df['Year'] >= 2015)]
```

> [!example]
> Пропущенные значения (`NaN` — Not a Number) при поэлементных операциях остаются `NaN` — не вызывают ошибку.

## Строковые операции

```python
df['Vehicle_Style'].str.lower()            # в нижний регистр
df['Vehicle_Style'].str.replace(' ', '_')  # замена символов

# Chaining — цепочка методов
df['Vehicle_Style'] = df['Vehicle_Style'].str.lower().str.replace(' ', '_')
```

> [!tip]
> Аксессор `.str` применяет строковые методы сразу ко всей Series. Методы можно объединять в цепочку (**chaining**).

## Агрегирующие операции и статистика

```python
df['MSRP'].min()    # минимум
df['MSRP'].max()    # максимум
df['MSRP'].mean()   # среднее
df['MSRP'].std()    # стандартное отклонение

df.describe().round(2)  # count, mean, std, min, percentiles, max для всех числовых колонок

df.nunique()            # количество уникальных значений по каждой колонке
df.isnull().sum()       # количество пропущенных значений по каждой колонке
```

> [!important]
> `.describe()` — быстрый способ получить статистику: count, mean, std, min, 25%/50%/75%, max.

## Группировка данных

```python
# Аналог GROUP BY в SQL
df.groupby('Transmission Type').MSRP.mean().round(2)
```

> [!example]
> Результат: средняя цена по типу трансмиссии.
> ```
> AUTOMATIC    30800.00
> MANUAL       29776.67
> ```
