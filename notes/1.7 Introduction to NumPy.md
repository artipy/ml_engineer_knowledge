# 1.7 Introduction to NumPy

> [!summary]
> **NumPy** — базовая библиотека для работы с массивами и вычислениями в Python. Стандартный импорт: `import numpy as np`.

## Создание массивов

| Функция | Описание |
|---|---|
| `np.zeros(size)` | Массив из нулей |
| `np.ones(size)` | Массив из единиц |
| `np.full(size, n)` | Массив, заполненный числом `n` |
| `np.array(list)` | Python list → NumPy array |
| `np.arange(start, end)` | Диапазон чисел (end — **exclusive**) |
| `np.linspace(start, end, size)` | `size` равноудалённых чисел от start до end (включительно) |

```python
np.zeros(5)        # [0. 0. 0. 0. 0.]
np.arange(3, 10)   # [3 4 5 6 7 8 9]
np.linspace(0, 1, 5)  # [0. 0.25 0.5 0.75 1.]
```

## Многомерные массивы

> [!tip]
> Передать размерность `(rows, columns)` или список списков.

```python
np.zeros((2, 3))          # 2×3 матрица из нулей
np.array([[1,2,3],[4,5,6]])  # из list of lists
```

## Индексация и срезы

```python
n[0, 0]    # элемент [строка 0, столбец 0]
n[1]       # вся строка 1
n[:, 1]    # весь столбец 1
n[:, 1] = 0  # перезаписать весь столбец
```

> [!important]
> Индексация начинается с нуля. Для 2D-массивов: `array[row, col]`.

## Генерация случайных чисел

```python
np.random.seed(2)               # фиксировать seed → воспроизводимость
np.random.rand(5, 2)            # uniform distribution [0, 1)
np.random.randn(5, 2)           # standard normal distribution
np.random.randint(0, 100, (5, 2))  # случайные целые [low, high)
```

> [!tip]
> Числа псевдослучайные (**pseudo-random**). `np.random.seed()` делает результаты **reproducible**.

## Поэлементные операции

```python
a = np.arange(5)   # [0 1 2 3 4]
a + 1              # [1 2 3 4 5]
a * 100            # [0 100 200 300 400]
(a * 2) + 10       # chain операций
a + b              # операция между двумя массивами одного размера
```

> [!example]
> В отличие от Python lists, не нужен цикл `for` — операция применяется ко всему массиву сразу.

## Операции сравнения и булева фильтрация

```python
a >= 2          # [False False True True True]
a > b           # поэлементное сравнение двух массивов

a[a >= 2]       # boolean filtering → [2 3 4]
a[a > b]        # только элементы, где условие True
```

## Агрегирующие операции

```python
a.min()   # минимум
a.max()   # максимум
a.sum()   # сумма
a.mean()  # среднее
a.std()   # standard deviation
```

> [!note]
> Возвращают одно число. Работают как с 1D, так и с многомерными массивами.
