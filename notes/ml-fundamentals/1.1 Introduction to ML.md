# 1.1 Введение в машинное обучение

## Что такое ML?

**Машинное обучение (ML)** — процесс выявления закономерностей в данных и использования этих закономерностей для предсказания будущих событий.

Компьютер обучается на примерах: получает данные → находит паттерны → делает прогнозы.

> [!example] Примеры задач ML
> - Предсказание погоды
> - Прогнозирование цен на акции
> - Результаты спортивных матчей
> - Оценка стоимости недвижимости

---

## Ключевые понятия

### Features (Признаки)

**Features** — информация об объекте, параметр которого мы хотим предсказать.

> [!example] Пример: предсказание цены дома
> Features: площадь, количество комнат, расположение, этаж...

### Target (Цель)

**Target** — то, что мы хотим предсказать.

> [!example] Пример: предсказание цены дома
> Target: цена дома

---

## Процесс обучения модели

```
Features + Target (известные данные)
        ↓
  Алгоритм ML находит оптимальные параметры
        ↓
     Обученная модель
        ↓
  Предсказание Target для новых данных
```

1. Предоставляем модели данные с известными **Features** и **Target**
2. Алгоритм анализирует данные и находит оптимальные параметры
3. Обученная модель делает прогнозы для новых данных, где Target неизвестен

---

## Разделение данных

| Выборка | Назначение |
|---|---|
| **Обучающая** (train) | Обучение модели |
| **Тестовая** (test) | Оценка качества модели на новых данных |

> [!important]
> Тестовая выборка — данные, которые модель **никогда не видела** во время обучения. Это обязательное условие честной оценки.

---

## Точность предсказаний

> [!warning] Важно понимать
> Предсказание — это **вероятность**, а не гарантия.

- Модель может ошибаться, особенно если обучающие данные не полностью отражают реальность
- Точность повышается при сравнении **средних значений** предсказаний и реальных значений
- Всегда важно оценивать качество модели

---

## Итог

ML — мощный инструмент для принятия информированных решений. Модель, обученная на данных, может делать предсказания на основе паттернов — но качество предсказаний всегда требует проверки.
